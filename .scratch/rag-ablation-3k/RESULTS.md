# RAG 방법 5종 ablation — 판정 스코어링 (3k 코퍼스)

현재 C 아키텍처에서 검색·읽기 경로의 레이어를 하나씩 쌓아 각 레이어의 기여를 판정관으로 잰다.

- **코퍼스**: 현재 3k 다양성 샘플(강화 임베딩 + 필터 컬럼)
- **질의**: 3k에 답이 존재하도록 작성한 근거 질의셋 21개(keyword 7 · semantic 7 · structured 7)
- **판정**: gpt-4o, 절대 적합도(0~3) + 전체 pairwise(10쌍, 순서 스왑·양방 일치만 승자)
- **파이프라인**: gpt-4o-mini(temp=0). ①~③은 raw 질의(검색 컴포넌트 격리), ④⑤는 QueryAnalyzer.
  나머지는 동일(같은 선택 단계, RAG_TOP_K=20).

| # | 방법 | 구성 |

## 해석 (정직하게)

1. **시맨틱(강화 임베딩)이 주력**: 시맨틱만으로 전체 2.43 — Route C 강화 임베딩의 위력 확증.
   **키워드만은 최약**(0.06 승률, semantic 계층 0.857) — 어휘 정확매칭 외엔 못 함.
2. **반직관: 하이브리드(raw) < 시맨틱만**(2.19 vs 2.43). raw 키워드를 더하면 오히려 손해 —
   한국어 토큰이 영어 상품명에 잡음으로 섞여 희석(structured 2.286→1.571). **키워드는 그냥 더하면 독.**
3. **분석기가 키워드를 살림**: ④에서 keyword 계층 3.0(한/영 검색어 생성). 단 semantic 계층은
   드리프트로 하락(2.43→2.14) — 분석기의 시맨틱 재작성이 직접 매칭을 놓치는 그 현상.
4. **반복이 최고**(2.571, 승률 0.274): 분석기가 깎은 **semantic 계층을 재시도로 복구**(2.14→2.57)
   하는 게 반복의 실제 가치. 승자매트릭스에서 ⑤가 전원에 우세(④는 근소·무승부 다수).
5. **다만 ⑤ vs ②(시맨틱만) 마진은 +0.14로 작음**(노이즈 범위 근처). full 파이프라인의 순증은
   시맨틱-only 대비 크지 않다 — 품질 천장은 코퍼스 커버리지·임베딩 품질에 있다.

## 요약

강화 임베딩(시맨틱)이 8할을 하고, **키워드는 분석기 없이는 독**, **분석기는 keyword↑·semantic↓
트레이드**, **반복이 그 트레이드를 메워 종합 최고**. 상대 순위(①최약 · ②강력 · ⑤최고)는 견고하나,
④⑤ 상위권 절대 마진은 노이즈에 가깝다.

⚠️ **한계**: n=21(계층당 7)이라 0.1~0.15 차이는 LLM 비결정성 노이즈. 상대 순위는 견고하나 작은
절대 마진은 참고. 통계력을 원하면 40~60 질의로 재측정.
